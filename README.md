
# Projeto Landing Page GTA

## Descrição

Este projeto é uma landing page para uma versão fictícia do jogo Grand Theft Auto V, com o tema de gatos. A página apresenta o jogo, suas plataformas e um botão de compra.

## Descrição Detalhada

A página é composta por:

*   **Cabeçalho:** Com o logo da Rockstar Games e do GTA, e um menu de navegação com as opções "Home", "Comprar" e "Suporte".
*   **Home:** Uma seção com uma imagem de fundo de um gato caracterizado como personagem do jogo, o título do jogo, uma breve descrição e um botão "Compre já".
*   **Comprar:** Uma seção com a capa do jogo, as plataformas em que está disponível e um botão para selecionar a plataforma.

O projeto foi desenvolvido com o objetivo de praticar a criação de landing pages com HTML, CSS e JavaScript.

## Tecnologias Utilizadas

*   HTML5
*   CSS3
*   JavaScript

## Como Utilizar

1.  Clone o repositório:
    ```sh
    git clone https://github.com/seu-usuario/projeto-gta-like.git
    ```
2.  Abra o arquivo `index.html` em seu navegador.
